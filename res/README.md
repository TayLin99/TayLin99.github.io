# 素材图片规范

## 格式要求

- **格式**: PNG（必须带透明通道 / alpha channel）
- **尺寸**: **2048 × 2048** 像素
- **色彩**: RGBA（32-bit）

## 说明

所有部件素材（衣服、头发、眼睛、配饰等）统一使用 **2048×2048 的画布**，部件内容放在画布中对应的位置，其余区域保持透明。

这样不同部件的图片直接叠加即可正确对位，无需额外的坐标/裁剪参数。

## 目录结构

```
res/
├── 背景/       # 背景图（可用 JPG，不需要透明通道）
├── 身体/       # 身体底图
├── 狗毛/       # 头发
├── 眉毛/
├── 眼睛/
├── 鼻子/
├── 鼻子装饰/
├── 腮红/
├── 嘴/
├── 衣服/
├── 手部配饰/
├── 配饰1/
├── 配饰2/
├── 配饰3/
└── 部位14/
```

每个部位文件夹内可以放多张 PNG 作为不同选项，文件名无特殊要求。
